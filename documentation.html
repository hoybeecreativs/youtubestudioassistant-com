üöÄ Getting Started

Installation

  1. Visit the Chrome Web Store
  2. Click "Add to Chrome"
  3. Confirm the installation
  4. The extension icon will appear in your browser toolbar

Initial Setup

  1. Click the extension icon in your browser
  2. Choose your preferred AI provider (Gemini is free!)
  3. Enter your API key (if using OpenAI or DeepSeek)
  4. Save your settings

‚ú® AI Content Generator

Generating Titles

  1. Open any video in YouTube Studio
  2. Look for the AI panel on the right side
  3. Click "Generate Title"
  4. Review and select your favorite option
  5. Click "Apply" to use it

Generating Descriptions

  1. In the AI panel, click "Generate Description"
  2. Add keywords or context (optional)
  3. Choose your preferred style (Professional, Casual, etc.)
  4. Click "Generate"
  5. Edit and apply the description

Generating Tags

  1. Click "Generate Tags" in the AI panel
  2. The AI will suggest relevant tags based on your video
  3. Select the tags you want to use
  4. Click "Apply" to add them to your video

üåç Translation Feature

Single Video Translation

  1. Open a video in YouTube Studio
  2. Navigate to the "Subtitles" tab
  3. Click "Translate" in the AI panel
  4. Select target languages (up to 110+)
  5. Click "Start Translation"
  6. Wait for the process to complete

Batch Translation

  1. Go to your YouTube Studio dashboard
  2. Click the extension icon
  3. Select "Batch Translation"
  4. Choose videos to translate
  5. Select target languages
  6. Click "Start Batch Translation"

üé® Thumbnail Generator

Creating Thumbnails

  1. Open a video in YouTube Studio
  2. Click "Generate Thumbnail" in the AI panel
  3. Choose a style or template
  4. Customize text and colors
  5. Preview and download
  6. Upload to your video

  • Content generation style

üîß Troubleshooting

Extension Not Working

  • Make sure you're on YouTube Studio (not regular YouTube)
  • Refresh the page
  • Check if the extension is enabled in Chrome
  • Try disabling other extensions temporarily

API Key Errors

  • Verify your API key is correct
  • Check if your API key has sufficient credits
  • Make sure the API key is active
  • Try regenerating your API key

Translation Issues

  • Ensure your video has a title and description
  • Check your internet connection
  • Try translating fewer languages at once
  • Wait a few minutes and try again
